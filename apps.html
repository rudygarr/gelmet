Apps

Tools that start with a real problem and stay close to the people using them.

Gelmet apps are not side projects looking for a purpose. They come from actual workflow pain, production pressure, and everyday organizational friction. The goal is not novelty. The goal is better work.

Current lineup

Two live-production tools are leading the way.

The first Gelmet apps are both rooted in the same instinct: if venues already have useful gear, the software should help them use that gear more effectively instead of forcing them into a giant technical leap.

Spot Projector

Turn a projector into a practical follow spot solution for rooms that need something simpler and more accessible.

Active build
Platform: iPhone + iPad Audience: schools, churches, community theaters Focus: fast setup and operator clarity

Spot Projector is for venues that need a spotlight effect without buying, storing, or staffing a traditional follow spot. It is designed to make setup testing, basic operation, and live use feel approachable enough for real people in real rooms.

SpotMover

A follow spot rethink for moving fixtures, built to make powerful lights feel more understandable in standalone use.

TestFlight
Platform: iOS + Mac path Audience: venues with movers, installs, production teams Focus: follow spot logic over lighting-console complexity

SpotMover is about meeting venues where they are. If they already own a capable fixture, the software should help them get useful, follow-spot-style control out of it without expecting every operator to think like a lighting programmer.

What makes these different

Built from real venue constraints.

These apps are not built from a blank-sheet startup fantasy. They come from real questions about budgets, staffing, room constraints, operator confidence, and gear that people already have on hand.
