# Gelmet

Gelmet is the public website for the studio and its apps.

## Publishing model

This repository is intended to stay simple:

- GitHub Pages publishes from `main`
- the site lives at the repository root
- `www.gelmet.com` is the primary domain

## Update workflow

1. Edit the site files in this repository
2. Commit the changes
3. Push to `main`
4. GitHub Pages republishes the site

## Main files

- `index.html`
- `apps.html`
- `consulting.html`
- `ai.html`
- `about.html`
- `contact.html`
- `styles.css`
- `assets/`
- `CNAME`

See `DEPLOYMENT.md` for the one-time GitHub Pages and GoDaddy setup.
